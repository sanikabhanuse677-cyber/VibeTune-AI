#  VibeTune AI

VibeTune AI is a visually interactive music recommendation web application that helps users discover songs based on their **mood** and **era preferences**. It features a modern glassmorphism UI, animated background, and a smooth audio preview system.

---

##  Features

  **Mood-Based Song Filtering** (Happy, Sad, Romantic, Calm)
  **Era Selection** (70s, 80s, 90s, After 2010)
  **Instant Audio Preview Player**
  **Add to Favorites (Local Storage)**
  **Remove & Manage Favorites**
  **Animated Live Background (Canvas)**
  **Modern Glassmorphism UI with Glow Effects**
  **Fast & Lightweight (Optimized for quick loading)**

---
